# Hotel Booking System

A simple hotel booking system written in Java.

## Overview

This project is a full-featured hotel booking system designed to manage room reservations, guest information, check-ins/check-outs, and payment processing. It is ideal for small to medium-sized hotels needing a reliable, extensible solution for daily operations.

## Features

- **Room Management**: Add, update, and remove rooms of different types.
- **Booking System**: Book rooms, modify bookings, and cancel bookings.
- **Check-in/Check-out**: Seamless guest registration and departure process.
- **Payment Processing**: Handle and process payments.

## Technologies Used

### Backend
- **Language:** Java
- **Frameworks/Libraries:** Spring Boot
- **Database:** MySQL
- **Build Tool:** Maven

### Frontend
- **Language:** JavaScript
- **Frameworks/Libraries:** React
- **Build Tool:** Node

## Getting Started

1. **Clone the repository:**
   ```sh
   git clone https://github.com/Poorna-Raj/hotel-booking-system.git
   ```
2. **Install dependencies:**
   
   - Frontend
   ```sh
   cd frontend
   npm i
   ```

   - Backend
   ```sh
   mvn clean install
   ```
     
4. **Configure the database:**
   > Update `.env` with your database configuration.
5. **Run the application:**
   ```sh
   mvn clean install
   mvn spring-boot:run
   ```

## Usage

- Visit the home page (URL or local endpoint)
- Register guests, book rooms, and manage hotel operations using the intuitive interface.
- Administrative users can manage room inventory.
